> Install diesel cli

`cargo install diesel_cli --no-default-features --features postgres`

> Run:
	cargo build

>To write post
	cargo run --bin write_posts

> To publish post
	cargo run --bin pusblish_post post_id_here

> To show posts
	cargo run --bin show_posts

> To delete a post
	cargo run --bin delete_post post_title_here
